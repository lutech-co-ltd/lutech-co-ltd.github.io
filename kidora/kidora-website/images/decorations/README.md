# Kidora 網頁裝飾元素

這個目錄包含 Kidora 網頁使用的裝飾元素，所有 SVG 都使用 Kidora 品牌色彩。

## 品牌色彩
- 珊瑚橘: `#f3a683`
- 蜜桃奶油: `#f8c291`
- 童趣藍: `#546de5`
- 粉嫩紅: `#ffb8b8`

## 裝飾元素列表

### 已生成的 SVG（共 9 個）
1. **star.svg** - 星星圖案（珊瑚橘/蜜桃奶油）
2. **question-mark.svg** - 問號圖案（童趣藍/粉嫩紅）
3. **speech-bubble.svg** - 對話泡泡（珊瑚橘）
4. **cloud.svg** - 雲朵圖案（蜜桃奶油）
5. **heart.svg** - 愛心圖案（粉嫩紅）
6. **sparkle.svg** - 閃爍星星（多色組合）
7. **rainbow.svg** - 彩虹圖案（多色漸層）
8. **lightbulb.svg** - 燈泡圖案（代表創意/智慧）
9. **book.svg** - 書本圖案（代表學習/知識）

## 使用方式

這些 SVG 可以直接在 HTML 中使用：

```html
<!-- 直接嵌入 -->
<img src="assets/images/decorations/star.svg" alt="Star">

<!-- 或作為背景 -->
<div style="background-image: url('assets/images/decorations/star.svg');"></div>

<!-- 或內聯 SVG -->
<svg>...</svg>
```

## 免費資源網站

如果需要更多裝飾元素，可以從以下網站下載：

1. **Flaticon** - https://www.flaticon.com
   - 大量免費圖示，可篩選 SVG 格式
   - 需要註冊，免費使用需標註來源

2. **Icons8** - https://icons8.com
   - 免費圖示庫，多種風格
   - 可下載 PNG/SVG

3. **Freepik** - https://www.freepik.com
   - 免費向量圖和插圖
   - 需要註冊

4. **Undraw** - https://undraw.co
   - 免費插圖，可自訂顏色
   - 適合裝飾元素

5. **Open Peeps** - https://www.openpeeps.com
   - 免費手繪風格插圖
   - 適合親子主題

## 自訂顏色

所有 SVG 都使用 Kidora 品牌色彩，如需調整：
1. 打開 SVG 檔案
2. 找到 `fill` 或 `stroke` 屬性
3. 修改顏色代碼
4. 調整 `opacity` 控制透明度
